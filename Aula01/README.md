## Aula 01 de Desenvolvimento de Software

# Descrição
Este código Java é parte da Aula 01 de Desenvolvimento de Software. Ele demonstra as operações básicas em Java, incluindo entrada de dados pelo usuário e saída de dados no console.

# Funcionalidades
Impressão de Texto: O programa começa exibindo o texto "Java - Aula 01".
Entrada de Dados: O programa solicita ao usuário que informe o nome do aluno.
Saída de Dados: Após a entrada de dados, o programa exibe o nome do aluno cadastrado.
